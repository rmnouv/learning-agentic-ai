```md

```

Example answer:

```md
Accuracy is not enough when the dataset is imbalanced. For example, in fraud detection or medical diagnosis, a model can have high accuracy but still miss the rare important cases. In this situation, precision, recall, F1-score, or ROC-AUC are more useful.

```

Important question:

```md
What is hallucination in LLMs?
```

Example answer:

```md
Hallucination happens when an LLM generates an answer that sounds plausible but is factually wrong or unsupported by the given context.
```

---

## Sunday — RAG System Design

Prepare this explanation:

```md
RAG means Retrieval-Augmented Generation. Instead of asking the LLM to answer only from its internal knowledge, we first retrieve relevant documents from a database, then give those documents to the LLM as context. This can reduce hallucination and make answers more grounded.
```

Know the pipeline:

```md
Documents
Chunking
Embeddings
Vector database
Retrieval
Prompt construction
LLM answer
Evaluation
```

## Saturday — From Notebook to Production

Review:

```md

Preprocessing pipeline
Model saving
API endpoint
Docker
Logging
Monitoring
Retraining
```

Interview question:

```md
How do you deploy a machine learning model?
```

Simple answer:

```md
First, I train and evaluate the model. Then I save the model and create an API, for example with FastAPI. I package the application with Docker, deploy it on a server or cloud platform, and monitor predictions, errors, and model drift.
```

```md

Feature leakage
Time-based split
Model drift

Reproducibility
```

Important idea:

```md
Data leakage is when information that would not be available in real life is used during training. It can make the model look very good during evaluation but fail in production.
```
